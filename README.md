# FurniSpace — Luxury Interior & Furniture Studio

FurniSpace is a high-fidelity, production-ready web platform for a premium interior design and luxury furniture boutique based in Dubai. The application focuses on brand storytelling, interactive product discovery, and a "High-Clarity Light" design aesthetic.

## 💎 Key Features

- **Animated Advertising Panel**: A premium, marquee-style scrolling ticker for real-time promotions and collections.
- **Dynamic Hero Section**: Interactive, full-screen slider with elegant typography and smooth GSAP-powered reveals.
- **Micro-Interactions**: Glassmorphic UI elements, smooth hover transitions, and professional navigation controls.
- **Interactive Catalog**: Category-specific browsing for Living, Bedroom, Dining, Outdoor, and more.
- **Product Detail Engine**: Rich modal views for product specifications, availability, and warranty details.
- **Zero-Emoji Policy**: A strictly professional, text-based industrial design language.

## 🎨 Tech Stack

- **Frontend**: HTML5 / CSS3 (Vanilla)
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- **Typography**: [Google Fonts](https://fonts.google.com/)
  - **Heading**: Cormorant Garamond (Serif)
  - **Body**: Inter (Sans-serif)
- **Icons**: Custom SVG system

## 🚀 Getting Started

Simply open `index.html` in any modern browser to view the storefront.

For development with hot-reload, you can use:
```bash
npx serve .
```
